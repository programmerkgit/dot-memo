Locking Script

アリスがボブのアドレスにビットコインを送金すると、そのビットコインには解除条件が与えられます。

その解除条件とは、「ボブのアドレスに対応した秘密鍵で生成した電子署名を提示した人のみがBTCコインを使用できる」というものです。

この解除条件をLocking Script*と言います。

Locking Scriptは「Script」という言語で書かれており、様々な形式の解除条件を記述することができます。

現在一番一般的なものがP2PKH(Pay to Public Key Hash)*というScriptで、上記のようにアドレスに対応した秘密鍵で解除出来るものです。

*他にもP2SH(Pay to Script Hash)という、任意のScriptを埋め込むことが出来る形式もあります。
