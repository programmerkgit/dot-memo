ビットコインのP2Pネットワーク

「ブロックチェーンの基本的な仕組み」で,ビットコインがP2Pネットワークで動いていることを説明しました。

もう少し詳しくビットコインのP2Pネットワークを理解していきましょう。

P2Pネットワークの特徴は、全てのネットワーク参加者が対等な形でネットワークに参加している事です。

世の中の典型的なサービスは、中央のサーバーデータを管理してサービスを提供し、それ以外の人はサービスを享受するという関係になります。

しかし、P2Pネットワークでは、全てのネットワーク参加者が対等であり、サービスの提供者であると共にサービスの利用者となります。

P2Pネットワーク上では権威的にサービスの管理を行う中央期間は存在せず、分散的な形でネットワークが成り立っています。

WHat Is Peer To Peer(p2p) Network

In the normal network, server and client

In the BitCoin network, there are no server. All Nodes have same role like a peer

All Nodes provide and consume services

It is a peer to peer network

P2P Characteristics

Resilient

If One node downs, Any Node in the network can serve service

Decentralized

There are no central server witch controls network arbitrarily

Open

Net work open, so anyone can access network data.
