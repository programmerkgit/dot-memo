ブロックチェーンのデータを保持するのがブロックチェーンデータベースの機能です。

全てのデータを保持しているフルブロックチェーンや、一部のデータしか持たないSPVノードなどが存在します。

マイニングノードはproof-of-workの計算をしています。

マイニングノードはフルノードである場合もあれば、SPVノードとして他のノードのデータを参照しながら稼働しているノードもあります。

Node Types and Roles

A BitCoin node may take take on different roles depending on the functionality

A BitCoin node is a collection of functions: routing, database, mining, and wallet

Full node with all four of these functionality is shown in Figure 8-1

Routing

To Participate in bitcoin network, node needs routing functionality. So every node in the network has routing functionality

Routing functionality validate and propagate transactions and blocks

Routing functionality discover and maintain connections to the network

Full Node

Full Node has full functionality, and has a copy of Full BlockChain. They can verify transaction authoritatively without reference to the other nodes.

SPV Node

Some Nodes maintain only subset of BlockChain, and verify transaction using a method calledsimplified payment verification or SPV. These nodes are known ad SPV or lightweight nodes.

Mining Nodes

Mining nodes compete to create new blocks by running speciallized hadware to solve the Proof Of Work algorithm. Some Mining Nodes mqy be full node, and others are lightweight nodes participating in (pool mining) and depending on mining pool server which maintains a full node
