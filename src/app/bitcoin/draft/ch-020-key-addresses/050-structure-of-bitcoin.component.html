楕円曲線暗号

ここで、楕円曲線上の2点に関して、足し算を定義します。

まずある二点(A,B)をとります。

その二点を結んだ直線と楕円曲線との交点をC`とします。

この交点は必ず1つだけとなることが知られています。

楕円曲線はx軸に対して対象なので、楕円曲線上に、C`とx軸について対象な点Cをとることができます。

この時、 A + B = C と定義することができます。

  • 省略事項: mod P, 結合法則, mod P でも同じ計算が成り立つ
  • y^2 (mod p) = x^3 + 7 (mod p)
  • 曲線上の点と点の足し算を定義する
  • BIPで定められた定点G
  • G + Gの足し算をランダムなk回するとする
  • 得られた座標Pk(x, y)のそれぞれをつなぎ合わせたものが公開鍵
  • kが秘密鍵
  • Kを与えられればコンピューターであれば計算は簡単にPkを計算できる。
  • Pkが分かっても、kの値が何であったかを知ることはできない。
