JavaScriptのDragイベント,Dropイベントには以下の種類があります。

  • drag
  • dragstart
  • dragend
  • dragenter
  • dragover
  • dragleave
  • dragexit
  • drop

それぞれのイベントのうち、ドラッグ対象で発生するイベントと、 ドラッグした対象をドロップされる側で発生するイベントに分けて理解するとわかりやすいです。

ドラッグ対象側で発生するイベント

  • drag: ドラッグを開始してから継続して発生
  • dragstart: ドラッグを開始時に発生
  • dragend: ドラッグが終了時に発生

ドロップされる側で発生するイベント

  • dragenter: ドラッグしたオブジェクトが要素の内側に入った時に発生
  • dragover: ドラッグしたオブジェクトが要素の上にある間継続して発生
  • dragleave: ドラッグしたオブジェクトが要素の外に出た時に発生

その他

以下はサポートされていません。

  • dragexit: 現在サポートされていないので省略

デモ

紫のエリアがドロップ可能エリアです。

数字はイベント発生回数をカウントしています。
