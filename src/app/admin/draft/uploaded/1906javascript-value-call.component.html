Javascriptの値渡し、参照渡しについては、以下のように考えると整理されます。

  • Javascriptはいつも値渡し
  • object型の場合は、objectへの参照値を値渡しする
  • 変数がobjectを参照している場合、プロパティーの変更は参照先のオブジェクトを変更する。

Object型とは

Primitive型以外はObejct型です。

Array, Function なども全てObject型です

Primitive型とは

以下のタイプがPrimitiveです。

大雑把には、数字と文字とnull系と覚えておけば良いでしょう。

  • Boolean
  • Null
  • Undefined
  • Number
  • BigInt
  • String
  • Symbol
