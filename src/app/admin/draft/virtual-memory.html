仮想記憶の管理

仮想記憶とは、実際の主記憶の容量よりも大きいアドレス空間を提供する仕組みです。

仮想アドレスと実アドレスの変換手法

仮想アドレスと実際のアドレスへの変換を行う必要があるのですが、そのために「動的アドレス変換機構(Dat: Dynamic Address Translation)」 が使用されます。

アドレスの変換をバイトやワード単位で実装すると、対応付けに利用するメモリが膨大となりアドレス空間を食いつぶしてしまいます。

そのため、アドレスの変換はある程度まとまったブロックの単位で実行します。

このブロックの大きさが固定の場合ページと呼び、サイズが可変の時はセグメントと呼びます。
